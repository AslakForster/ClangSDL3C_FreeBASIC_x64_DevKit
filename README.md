ClangSDL3C_FreeBASIC_x64_DevKit v1.1.0 is a development kit for 64-bit Windows. It provides an easy setup for C programming (using the Clang C compiler) and for compiling FreeBASIC projects, both featuring SDL3 support. The kit includes some example code. The kit also features the Mini-IDE Launcher v1.1.0 for ease of use and is released under the Permissive Non-Sale License (PNSL). 

Press F5 to compile. A CMD window opens and can be closed after compilation.

Launcher with C and FreeBASIC editors, both featuring SDL3 support, built using Rust and egui.

Version 1.1.0 was released 2025-10-29.
